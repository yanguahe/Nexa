# `nexa setup`

Initialize `~/.nexa/nexa.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
nexa setup
nexa setup --workspace ~/.nexa/workspace
```

To run the wizard via setup:

```bash
nexa setup --wizard
```
