# `nexa tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
nexa tui
nexa tui --url ws://127.0.0.1:18789 --token <token>
nexa tui --session main --deliver
```
